### 数组和链表时间复杂度对比

| 操作    | 数组 | 链表 |
| ------- | ---- | ---- |
| prepend | O(n) | O(1) |
| append  | O(1) | O(1) |
| lookup  | O(1) | O(n) |
| insert  | O(n) | O(1) |
| delete  | O(n) | O(1) |
